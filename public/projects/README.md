## File tips

- **Formats:** `.png`, `.jpg`, `.jpeg`, or `.webp` вҖ” use the `1` / `2` / `3` basename (e.g. `1.png`).
- **Recommended size:** 1280Г—720 or wider (16:9). The UI crops with `object-cover`.
- Missing files show a colored placeholder until you add them.
